SESSION DESCRIPTION:
Recently research community consumes computational intelligence in every possible
field, like healthcare, automobiles, agriculture, telecommunication, automation, robotics,
retail industry, banking etc to leverage its amazing power. In an expanding world with
limited resources and increasing complexity, computational intelligence becomes a
necessity. Computational intelligence can offer new solutions to effectively make
complexity manageable. This session on “Expansions in Computational Intelligence and
Application” is aimed to provide a platform for researchers, professionals, academicians
and students to present and discuss leading research topics, developments and future
directions in this area. Prospective authors are invited to submit their full papers in this
special session that have not been yet submitted or published anywhere.

RECOMMENDED TOPICS:
Topics to be discussed in this special session include (but are not limited to) the following:
Multidimensional artificial intelligence
Machine vision and information exchange
Machine, Deep Learning
AI based Techniques and Algorithms for Build an Intelligent System
Intelligent Scheme for Information/Network Security using Machine/Deep Learning
Big data visualization, modelling, and analytics
Healthcare Automation using Machine / Deep Learning
Artificial Intelligence and Pattern Recognition
Adaptive Wireless Communication
Cloud based applications for Machine Learning
Security Technology and Information Assurance
Intrusion detection and prevention
Optimization approaches
Advanced and multidimensional networks (including WSNs, MANETs, VANETs,P2P)
Security and privacy in wireless networks
Network security policy, theory, and tools
